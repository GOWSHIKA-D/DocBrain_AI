# 🧠 DocBrain – AI Government Document Simplifier

DocBrain is an AI-powered web application that helps users easily understand complex government documents such as notices, schemes, and forms. It simplifies legal language into plain text, summarizes long PDFs, and allows users to ask questions about the document.

---

## 🚀 Live Demo
🔗 https://docbraingov.lovable.app/

---

## 📌 Features

- 📄 Upload PDF or scanned document images
- 🤖 AI-based document summarization
- 💬 Chat with document (ask questions directly)
- 🧾 Simple language explanation of complex content
- 🔍 Extract key details like eligibility, dates, and instructions
- 🌐 Multilingual support (optional future enhancement)
- 🔊 Read-aloud feature (future enhancement)

---

## 🎯 Purpose

DocBrain is designed to:
- Help citizens understand government schemes easily
- Reduce confusion in legal and official documents
- Support students and rural users with simple explanations
- Improve accessibility of public information

---

## 🏗️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask / Node.js - based on implementation)
- AI Integration: NLP / LLM APIs
- Document Processing: PDF parsing libraries
- Deployment: Lovable / Netlify / Render

---

## ⚙️ How It Works

1. User uploads a PDF or image document
2. System extracts text from the file
3. AI model processes and simplifies the content
4. User can:
   - Read summary
   - Ask questions
   - Get key insights instantly
